ownCloud has been installed.

For production, configure:
  - owncloud.domain and owncloud.trustedDomains for the public hostname
  - owncloud.admin.existingSecret or owncloud.admin.password for the initial admin credentials
  - database.existingSecret or database.password for the database credentials
  - persistence.data.size and storageClassName for file storage

Admin credentials are only applied when ownCloud initializes a fresh data volume.
