Reusable charts for Kubernetes applications.

Small, practical Helm charts with clean defaults, versioned releases, and support for common homelab and production-style deployment patterns.

Install from Helm View source 
# Add the repository
helm repo add harish2k01 https://harish2k01.github.io/helm-charts
helm repo update

# Install BentoPDF
helm install bentopdf harish2k01/bentopdf

# Install Firefly III
helm install firefly-iii harish2k01/firefly-iii

# Install with custom values
helm install bentopdf harish2k01/bentopdf -f values.yaml

# Show all published versions
helm search repo harish2k01 --versions

Helm native

Published with an index.yaml and packaged chart archives so users can install with normal Helm commands.

Portable defaults

Public chart defaults avoid personal domains, namespaces, storage classes, and cluster-specific controllers.

Versioned releases

New chart versions get GitHub Releases with notes and packaged chart archives attached.
